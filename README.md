# PKI Project

## Overview
This is a PKI project for managing certificates and keys using Go.

## Structure
- **cmd/**: Main application.
- **pkg/certmanager/**: Certificate and key management.
- **pkg/certgeneration/**: Certificate generation.
- **pkg/certexpiry/**: Certificate expiry management.
- **pkg/blacklist/**: Blacklisting certificates.
- **pkg/expired/**: Handling expired certificates.
- **pkg/extend/**: Extending validity of expired certificates.
- **scripts/**: Scripts for generating certificates.
- **config/**: Configuration files.
- **docs/**: Documentation.
